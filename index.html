jade-press

cms based on mongodb, nodejs, koa, vue and more

githubdownload

try it

#for mongodb driver ubuntu (optional)
#sudo apt-get install libkrb5-dev
#or visit https://github.com/mongodb/node-mongodb-native#troubleshooting for more

#for canvas ubuntu (optional)
#sudo apt-get install libcairo2-dev libjpeg8-dev libpango1.0-dev libgif-dev build-essential g++
#visit https://www.npmjs.com/package/canvas for more platform

git clone git@github.com:jade-press/jade-press.git
cd jade-press
npm install
bower install
cp config-sample.js config.js

#then edit config.js yourself to config

#install plugins
gulp install

#run it
node app

#then visit http://127.0.0.1:7200 (if use default setting)

features

  • based on mongodb(database and session), nodejs 4+, koa...
  • built in jade(pug) editor
  • file upload and insert
  • custom routes
  • plugin system
  • theme system
  • user system
  • user groups and custom access control based on url
  • post and category
  • custom style & script for every post
  • can be used as a module to extend

use jade-press as a module

visit https://github.com/jade-press/blog.jade-press.org as a example
